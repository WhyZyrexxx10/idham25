Siapa saya?

Saya adalah seorang administrator sistem Linux dengan lebih kurang dari 2 tahun pengalaman dalam mengelola dan mengoptimalkan infrastruktur berbasis Linux. Saya mewakili provinsi Bengkulu dalam kegiatan lomba kompetensi siswa (NASIONAL) smk pada tahun 2024 pada bidang IT NETWORK SYSTEM ADMINISTRATION.

Saya mengkhususkan diri dalam administrasi server Linux, kontainerisasi dengan Docker dan Kubernetes, infrastruktur cloud di AWS, dan otomasi menggunakan Ansible dan skrip Bash. Saya telah berhasil memigrasi beberapa organisasi dari sistem proprietary ke Linux, menghasilkan penghematan biaya yang signifikan dan peningkatan kinerja.

Ketika saya tidak bekerja dengan sistem Linux, saya suka berkontribusi pada proyek sumber terbuka, menulis artikel teknis, dan mentoring administrator sistem junior.
